# Sistema de Analise Econometrica

Sistema completo de analise econometrica desenvolvido em Streamlit Python.

## Funcionalidades

- Autenticacao de usuarios
- Upload de arquivos CSV e XLSX
- Sugestao automatica de modelos
- Multiplos modelos econometricos:
  - Regressao Linear (OLS)
  - Regressao Logistica (Logit)
  - Modelo Probit
  - ARIMA (Series Temporais)
  - VAR (Vetores Autoregressivos)
  - Random Forest
- Visualizacoes completas
- Exportacao em PDF e PNG

## Instalacao

```bash
cd econometric_system
pip install -r requirements.txt
```

## Executar

```bash
streamlit run app.py
```
